# ESP32 RF Gateway Pro

Professional 433 MHz RF Gateway for ESP32 featuring real-time RF monitoring, signal learning, replay, OTA firmware updates and a modern web interface.

---

## Overview

ESP32 RF Gateway Pro is a complete RF development and testing platform designed for ESP32.

Using a standard 433 MHz transmitter and receiver pair, the gateway can capture, analyze, store and retransmit RF signals directly from a web browser without reconnecting the device to a computer.

The project was developed for makers, developers and IoT enthusiasts who need a reliable RF learning and replay solution.

---

## Key Features

* 📡 Live RF Monitor (WebSocket)
* 📥 RF Signal Learning
* 📤 RF Signal Replay
* 💾 Save RF Codes (LittleFS)
* 🔍 Search and Delete Stored Codes
* 🎲 Random RF Code Generator
* 📂 Backup & Restore
* 📶 WiFi Configuration Portal
* 🔄 OTA Firmware Updates
* 🌐 Responsive Web Dashboard
* ⚡ Fast WebSocket Communication

## Hardware Requirements

* ESP32 Development Board
* 433 MHz RF Receiver
* 433 MHz RF Transmitter

### Tested Modules

Receiver

* XY-MK-5V
* SYN480

Transmitter

* FS1000A
* SYN115

## Default GPIO Configuration

| Function       |   GPIO |
| -------------- | -----: |
| RF Transmitter | GPIO12 |
| RF Receiver    | GPIO13 |
| Status LED     |  GPIO2 |

GPIO assignments can be changed in the configuration file.
